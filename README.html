Plugin Booster Installation Readme


What's inside this folder

  1. resources.qrc : file to be compiled when updating the UI. Contains the informations/objects of the UI created in QTCreator
  2. resources.py : results of the compilation to be read by QGIS
  3. __init__.py : allows the script to be loaded as a plugin by QGIS
  4. Booster_dialog_base.ui : contains graphical elements of the window, may be modified with QTCreator
  5. Booster_dialog_base.py : Loads the window elements in python
  6. booster.py : Main file
  7. icon.png : plugin icon displayed in QGIS
  8. qgis_style_files : folder containing style files to be loaded for the manipulated layers in qgis
  9. python_files : Python classed used by this program

How to install

  1. If resources.py is not present in your plugin directory, compile the resources file using pyrcc5 (simply use pb_tool or make if you have automake)
  2. Install missing packages
  3. Copy the entire directory containing your new plugin to the QGIS plugin directory (see Notes below)
  4. Test the plugin by enabling it in the QGIS plugin manager
Notes:
  • You can use QGIS plugin reloader extension to automatically compile, deploy, and reopen your plugin.

For information on writing PyQGIS code, see http://loc8.cc/pyqgis_resources for a list of resources.
